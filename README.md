# SpringBoot人事管理系统 java1190

## 项目概述

SpringBoot人事管理系统是一个基于前后端分离架构的开源项目。它主要用于企业中的人事管理，包括员工信息管理、部门管理、薪资管理等模块。

## 技术栈

### 后端

- **SpringBoot**
- **MyBatis**
- **MySQL**

### 前端

- **Vue.js**

### 推荐环境配置

- **IDEA**
- **JDK 1.8**
- **Maven**
- **MySQL**

## 功能模块

- 员工信息管理
- 部门管理
- 薪资管理

## 系统角色

- 系统管理员
- 普通员工

## 运行环境

- 后端：遵守推荐环境配置
- 前端：支持现代浏览器

## 部署步骤

1. 准备好相应的开发环境
2. 克隆或下载项目代码
3. 导入数据库文件
4. 编译并运行后端服务
5. 前端项目独立部署
